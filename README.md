# Chrome Dino RL

Небольшой Python-проект с визуализацией обучения агента играть в игру Chrome Dino.  
Агент использует Q-learning: он наблюдает состояние динозавра и препятствий, выбирает действие — прыгать или ничего не делать — и постепенно улучшает стратегию.

## Что внутри

- `app.py` — Streamlit-приложение с интерфейсом, игрой, обучением и графиком прогресса.
- `dino_game.py` — логика игры Chrome Dino.
- `rl_agent.py` — Q-learning агент.
- `requirements.txt` — зависимости проекта.

## Как запустить

1. Создать виртуальное окружение:

```bash
python3 -m venv venv


2. Активировать его:

source venv/bin/activate

3. Установить зависимости:

pip install -r requirements.txt


4. Запустить Streamlit-приложение:

streamlit run app.py

После запуска Streamlit откроет приложение в браузере. Обычно адрес будет:

http://localhost:8501

Использование
В интерфейсе можно запустить обучение агента, выбрать количество эпизодов, наблюдать за игрой и смотреть график прогресса обучения.
